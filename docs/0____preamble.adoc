This layered standard on top of FMI 3.0, defines how to describe and simulate network signals as input and output variables of an FMU.

NOTE: Note: Although the document refers to version 3.0 of the FMI standard, everything described in this document also applies to all subsequent minor versions.
For further information on compatibility, see section https://fmi-standard.org/docs/3.0/#VersioningLayered[Versioning and Layered Standards] in the FMI 3.0 specification.
